*CLASS ASSIGNMENT ON ESSAY & HTML USING OL & UL WITH IMAGE AND VIDEO USING ANCHOR TAG*

  • A hyperlink, also called a link or web link, contains an address for a destination and acts as a reference to data. A user can easily follow, jump to, and be directed to the destination by either clicking,
    tapping on, or hovering over the link. In HTML, we can use the image element to add images & video element to add video on the page.

    In this class assignment, we are adding an image of three cats and a single video. If we wanted to make that image & video a clickable link, then we can place it inside a set of anchor tags.
    We can also add the target="_blank" attribute to have that link open up in a new tab. Also, the video can be on Alto-Play, Loobs, Muted or Control while we can
    also place our date using SUB TAG.
